# ISWA.jsx

![reactiontime](./reactiontime.png)

# Install
```
npm install -S @iswa/jsx
```

# CRA Example
```
import React from 'react';
import ReactDOM from 'react-dom';
import { S10000 } from '@iswa/jsx'

ReactDOM.render(
    <S10000 />,
  document.getElementById('root')
);
```
